1. Rename app.apk to app.zip file

2. Download and extract dex2jar from https://sourceforge.net/projects/dex2jar/files/ in a folder.

3. Extract app.zip and copy classes.dex file to dex2jar folder

4. Run command "./d2j-dex2jar classes.dex"

5. classes-dex2jar.jar will be created

6. View files in jar using JD_GUI
