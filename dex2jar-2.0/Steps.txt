1. Rename app.apk to app.zip

2. Download and extract dex2jar from https://sourceforge.net/projects/dex2jar/files/

3. Extract app.zip and copy classes.dex from app folder to dex2jar folder

4. Run "./d2j-dex2jar classes.dex"

5. classes-dex2jar.jar file will be created.

6. Open jar file using JD-GUI
